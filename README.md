# GIT HUB
Instalar git hub y crear un archivo notas y un README.md.
 
## Instalación
1. Descargar de la página oficial.
2. Abrir la terminal (Git Bash).
3. Colocar su nombre, email y dar enter.
 
## Tecnologías utilizadas
- VS Code
- Git Hub
- Git Bash
